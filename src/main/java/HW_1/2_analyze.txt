В заголовке сайта справа сверху есть кнопки с регистрацией или входом в аккаунт. В левом верхнем углу находится логотип
сайта. Под логотипом слева находится вертикальный блок, содержащий подпункты: "Заглавная страница", "Участие",
"Инструменты", "Печать/экспорт", "В других проектах", "На других языках". В каждом подпункте есть свои уникальные элементы.
Ниже кнопок с регистрацией есть поле для поиска. Ниже поля для поиска есть блок с приветственной надписью,
содержащий информацию о количестве статей и предлагающий создать статью.
Ниже этого блока есть блоки "Избранная статья", "Из новых материалов", "Хорошая статья", "Текущие события",
"изображения дня", "В этот день", "Совместная работа недели". Каждый из этих блоков содержит информацию и ссылки.
Ниже этих блоков, выше подвала сайта, расположены ссылки в виде иконок на другие проекты.
В подвале сайта дается общая правовая информация о том, какая лицензия используется, политика конфиденциальности,
описание сайта, переход на мобильную версию, статистика, условия использования.
